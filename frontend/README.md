# Legal Lens Frontend

This is the user interface for the Legal Lens Clause Simplifier. 
It connects to the fine-tuned Gemma-2b API hosted at [Agent-Omkar/legal-lens-api](https://huggingface.co/spaces/Agent-Omkar/legal-lens-api).

## Setup
Point your `API_URL` in `app.py` to your deployed backend space.
